Post Builder.

How to use this app

This app helps you write Facebook posts that work well when Facebook cross-posts them to Instagram. The trick is putting the image description inside the post itself, instead of in Facebook's alt-text box — that way it doesn't get stripped when the post crosses to Instagram.

If you attach more than one photo, each photo gets its own description box. Switching photos wipes the boxes so old text never ends up on the wrong picture.

The Mentions list is a place to keep names beginning with @ for quick use. Saved names are not added to your post. Tap a name to copy it to the clipboard, then paste it in Facebook where you want it. (Facebook only makes a real tag when you type @ inside Facebook and pick a name from its suggestion list — pasted text stays as plain text.) Tap "Edit list" to remove names you no longer need.

When you're ready, tap "Share to…". iOS opens its share menu and your text is also copied to the clipboard at the same moment. If Facebook only takes part of the text, paste into the Facebook post box and the full post drops in. Hashtags become clickable when the post goes live; mentions stay as plain text.

Drafts, saved names and hashtag lists live on this phone. The Backup section at the bottom saves a copy you can keep in iCloud Drive or Files. If anything is ever lost, restore it from that file.
